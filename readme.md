# @kottetall/testdata.nu

A lightweight npm package for generating **random, fake Swedish test data** for development and mocking purposes.

This package is a simplified ("lite") version of https://testdata.nu, focused on the most commonly used generators such as **Swedish personal identity numbers (personnummer)**, names, and other basic identity-related data.

> ⚠️ **Disclaimer:** All generated data is completely fictional and intended for testing only.  
> Do **not** use this package for production identity handling, validation, or any real-world identification purposes.

---

## Features

- Generate fake **Swedish personnummer**
- Generate fake **names**
- Generate randomized Swedish test identities
- Designed for developers who need mock data quickly
- Lightweight and easy to use

---

## Installation

```bash
npm install @kottetall/testdata.nu
```
